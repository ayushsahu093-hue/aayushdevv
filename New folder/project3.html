MAAZA.

A liquid-smooth interactive journey through flavor.

120 FPS Fluidity
50% More Engagement
0.2s Interaction Lag

The Vision

Maaza wanted to translate the "richness" of their fruit drinks into a digital space. We aimed to create a scrollytelling experience that feels as vibrant and refreshing as the product itself.

The Execution

Engineered using Next.js for a robust foundation and Framer Motion for imperative scroll-mapping animations. The UI leverages Tailwind CSS for rapid, utility-first styling and Lucide React for a consistent, beautiful icon set.
